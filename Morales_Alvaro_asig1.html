Información de HTML

¿Qué es?

El HTML (Lenguaje de Marcado de Hipertexto) es la tecnología utilizada para la estructuración de una página web y sus contenidos. A pesar de que en su nombre se menciona la palabra "lenguaje", este no es un lenguaje de programación debido a que no puede crear una funcionalidad dinámica; es un lenguaje de marcado (markup Language) que define la estructura de su contenido y con el cual se pueden crear y estructurar secciones, párrafos y enlaces usando elementos, etiquetas y atributos.

Con el tiempo han llegado a salir distintas versiones de HTML. En cada versión se incluían nuevas etiquetas, siendo 18 de la primera versión a unas casi 150 etiquetas hoy en día. Con las etiquetas se puede lograr que una imagen o palabra te envíen a otro sitio por medio de un hipervinculo o También se puede hacer que los textos se vean en negrita, cursiva, tachado, sombreado, etc...


Estructura de un elemento HTML

Las partes principales del elemento de HTML son los siguientes:

  1. La etiqueta de apertura: Consiste en el nombre del elemento (en este caso, p), encerrado por paréntesis angulares (< >) de apertura y cierre. Establece dónde comienza o empieza a tener efecto el elemento —en este caso, dónde es el comienzo del párrafo—.
  2. La etiqueta de cierre: Es igual que la etiqueta de apertura, excepto que incluye una barra de cierre (/) antes del nombre de la etiqueta. Establece dónde termina el elemento —en este caso dónde termina el párrafo—.
  3. El contenido: Este es el contenido del elemento, que en este caso es sólo texto.
  4. El elemento: La etiqueta de apertura, más la etiqueta de cierre, más el contenido equivale al elemento.
